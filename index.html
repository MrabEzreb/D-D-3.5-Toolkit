So yeah, basically, it is supposed to be DnD, but on a computer! No more messing with grid paper!

No more messing with grid paper!

Basically, the main component of DnD is messing with the grid paper, for maps, for descriptions, for everything. This is handled by the DnD toolkit by putting you into first person mode INSIDE your DnD world!

Built using java.

Yes, that's right, this is built on java, the same programming language used for that popular game "Minceraft." We even use some of the same apis, such as JSON (JavaScript Object Notation) and Java3D.

Full Campaign Support

Yes, you read that right, FULL CAMPAIGN SUPPORT! That means, not only can you have your dungeons on here, you can have your towns, cities, kingdoms, and universes all inside of here, AND connect them.

Automatic Updating

Yup, auto-updating. A: when you run the Toolkit for the first time, it downloads the most recent release. Then, every time after that, it will automatically update for you!
